Constellation Action Keyboard Shortcut User Action Menu Icon
Run Horizontal Line Arrangement Ctrl + Alt + H Arrange -> Horizontal Line
Run Vertical Line Arrangement Ctrl + Alt + V Arrange -> Vertical Line

The line arrangements arrange all the nodes of the graph into either a horizontal or vertical line. The horizontal line orders nodes across the x-axis and the vertical line orders nodes across the y-axis. Nodes are ordered by their internal node id's.
